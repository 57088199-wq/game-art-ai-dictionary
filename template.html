万能替换模板

一、角色替换

保留姿态，换人物
保留原有构图、动作姿态、光影和透视，仅替换角色为（新角色），面部特征清晰，人体结构准确，细节丰富，画质清晰。
只换脸，不换人
保留人物姿势、服装、场景和光影，仅替换面部为（新脸型 / 风格），五官精致，比例协调，画质高清。
换性别 / 年龄
保持动作、场景、构图不变，将人物替换为（男 / 女 / 少年 / 老年），风格统一，光影自然，细节完整。
换服装 / 造型
保留人物五官、姿态、场景不变，替换服装为（风格 / 类型），发型同步更换，整体协调，质感真实。

二、场景替换

只换背景，保留人物
保留前景人物、动作、光影和构图，移除原有背景，替换为（新场景），景深合理，氛围统一，细节丰富。
室内 ↔ 室外互换
保持人物、姿态、构图不变，将场景替换为（室内 / 室外 / 城市 / 自然），光影适配新环境，透视正确。
风格化场景替换
保留主体不变，背景替换为（赛博朋克 / 古风 / 科幻 / 梦幻）场景，色调统一，电影级质感，高清细节。

三、风格 / 画质替换

换画风
保留人物、构图、动作不变，整体画风替换为（写实 / 动漫 / 插画 / 胶片），色彩和谐，线条流畅。
换光影色调
保持主体不变，替换光影为（柔光 / 硬光 / 暖光 / 冷光 / 电影光），氛围感强，色调统一。
通用高清强化
超高分辨率，细节丰富，质感真实，光影自然，构图完美，无瑕疵，专业级画质。
